# Tester Invite Messages

Copy/paste these into iMessage when sending invites. Three versions for
different audiences. Edit names where indicated.

## Version 1: For your son's teammates (his friends)

```
Yo, [son's name] and I have been working on a basketball training app
for hoopers like us. It gives you daily workouts, tracks your stats,
and gives you badges for hitting milestones.

Want to try it? It's free, takes 30 seconds to set up.

→ https://hooperspath.com

Best on iPhone — Sign in with Apple, then add it to your home screen
(Share button → Add to Home Screen).

Let me know what you think — what's good, what's broken, what would
make it actually useful for you. We're trying to make this real.
```

## Version 2: For parents (more formal)

```
Hi [parent name],

[my son's name] and I have been building a basketball training app
designed for high school players. It includes daily structured workouts
(no gym needed), tracks personal records and game stats, and helps
players build training consistency.

We're testing it with a small group before launching publicly. Would
[their kid's name] be interested in trying it?

→ https://hooperspath.com

It's free, no ads, and works on iPhone (add to home screen for the
best experience). Designed for ages 13+.

Happy to chat or answer any questions.
```

## Version 3: For coaches

```
Coach [name],

I'm building a training app for high school basketball players —
daily workouts, drill library with proper diagrams, game tracking,
PR tracking. Designed to be done in 30-45 min/day.

Would love your eyes on it before I share more widely. Free to try,
no signup beyond Sign in with Apple.

→ https://hooperspath.com

If it's useful for your players, I can show you how to create a team
inside it so you can see their training (we built a basic coach view).

Any feedback would help us build it right.
```

## Follow-up messages (use after 3-7 days)

### If they signed up but haven't returned:
```
Hey, did you get a chance to try the app? Even 30 seconds of feedback
helps — what stopped you from using it more?
```

### If they're using it and engaged:
```
Glad it's working for you! Two quick questions:
1. What's the one thing that's most useful?
2. What's one thing missing that would make it 10x better?
```

### If they hit a bug:
```
Sorry about that — can you screenshot what you saw? It helps me
fix things quick. The Settings tab also has a "Send Feedback" button
that includes diagnostic info.
```
